<h2>Getting Started</h2>

<h3>Prerequisites</h3>
<ul>
  <li>Dart SDK (2.x or later)</li>
  <li>Basic programming knowledge (optional)</li>
</ul>

<p>
Download Dart SDK from:
<a href="https://dart.dev/get-dart">https://dart.dev/get-dart</a>
</p>

<h3>Clone the Repository</h3>

<pre>
git clone https://github.com/nazmulcse247/dart_tutorial.git
cd dart_tutorial
</pre>

<h3>Run Dart Examples</h3>

<pre>
dart run bin/your_file_name.dart
</pre>
